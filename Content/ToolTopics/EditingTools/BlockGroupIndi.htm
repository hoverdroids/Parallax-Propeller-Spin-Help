Block-Group Indicators

Sometimes it may be hard to see exactly how groups of code are logically arranged simply by their level of indention.  The Propeller Tool can optionally indicate the logical block-groups of conditional blocks or loop blocks as shown below.  To toggle this feature on or off, press Ctrl + I.  Or, you may check and uncheck the Show Block Group Indicators box under the Preferences - Appearance Tab in the Preferences Dialog

Block-Group Indicators

 

Note that only compilable code that is actually within a conditional block or a loop block is actually enhanced with the indention indicators.  Also, this is simply a visual aid to see how the code will be executed; it does not affect the code or the source file physically; only the actual levels of indention do that. See Indenting and Outdenting for more information.
