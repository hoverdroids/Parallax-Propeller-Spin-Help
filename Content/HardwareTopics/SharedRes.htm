Shared Resources

There are two types of shared resources in the Propeller:

  1. Common: these can be accessed by any number of cogs at any time. These include:
    • I/O Pins
    • System Counter
  1. Mutually Exclusive: Access to these is gained through the Hub. These resources can also be accessed by every cog, but only by one cog at a time. These include:
    • CLK Register
    • Main Memory
    • Locks (Semaphores)
