Hardware Connections

This example wiring diagram provides host and EEPROM access to the Propeller chip.  Here, the host access is achieved through the Propeller Plug device (a USB to TTL serial converter).

Typical Hardware Connection (with Propeller Plug USB-to-Serial adapter)

Example wiring diagram that allows for programming the Propeller chip and an external 32 Kbyte EEPROM, and running the Propeller with an external crystal.  The Propeller Plug is part# 32201; available from www.parallax.com.

 

Alternative Hardware Connection (with standard DB-9 serial port)

Alternative example for programming the Propeller chip without the Propeller Plug. Propeller chip not shown.
