What's New

Each release of Propeller Help and the Propeller Tool software can include new or enhanced items that may not be readily apparent. This topic is a summary of these notable changes.

Propeller Help

Version 1.3 (released with Propeller Tool Software v1.3.2)

  • Updated Questions and Answers to include information on method and object reference limitation in Spin objects.

Version 1.2 (released with Propeller Tool Software v1.3)

  • New Questions and Answers system that provides a comprehensive overview of the Propeller P8X32A multicore microcontroller.
  • New Coding Techniques articles Stack Space and Code Execution Time.
  • New DB9 Serial to Propeller programming schematic in Hardware Connections; this is now the recommended method for DB9-based serial connections.
  • Updated Propeller Education Kit Labs: Fundamentals text and example code to version 1.2; the source code is available in the Propeller Tool from File > Open From > PE Kit - Examples.
  • Fixed minor typos in the Spin Tutorial, most notably an incorrect object call in Spin Exercise 7-1.
  • Updated Propeller Tool shortcut key tables.
  • Updated Propeller Tool menu items and descriptions.

Version 1.1 (Released with Propeller Tool Software v1.2.6)

This is the first official release of a much-improved help system in the Propeller Tool software. It coincides with the release of Propeller Manual v1.1.

  • The Using the Propeller Tool chapter from Propeller Manual v1.0 was enhanced and included in Propeller Help so it follows Propeller Tool software changes more readily.
  • The Propeller Programming Tutorial chapter from Propeller Manual v1.0 was enhanced and included in Propeller Help. This information will be further enhanced in the near future.
  • All critical information tables formerly only in the Propeller Manual are now included here.
  • Includes Why the Propeller Works, a letter from the President of Parallax describing the nature of the Propeller chip's design.
  • Contains hundreds of enhancements to previous information, links to Propeller resources, and other useful optimizations for on-line help.

Version 1.0

The first release of Propeller Help.

 

Propeller Tool Software

Version 1.3.2

General

  • Enhanced editor's undo and redo operations to track cursor movements and recreate them. This helps with performing undo/redo operations that are outside of the current view by scrolling the view to that region (to show the cursor just before or after the soon-to-be-affected item)... the next undo/redo affects that item.

  • Added range limitation code and range indication to Preferences' Post-Reset Delay item.
  • Enhanced Library Warning dialog and underlying code to be more clear and to function properly with source locating code.
  • Updated download code to handle unexpected received data during reset of Propeller, such as what happens with the QuickStart board because of the extra buffer component.

Bug Fixes

  • Fixed bug that prevented "ignore temporarily" option from working when a library folder is missing.

Library

  • Updated Clock to v1.2.
  • Updated FullDuplexSerial to v1.2.1.

Library Demos

  • Added Clock Demo v1.0.
  • Added FullDuplexSerial_Test1 v1.1.
  • Added FullDuplexSerial_Test2 v1.2.

Misc

  • Updated Parallax USB Driver Installer (USB to Serial) to v2.08.24.

 

Version 1.3

General

  • Added /NewInstance command-line switch that forces a new instance regardless of the SingleInstance registry setting. This is used to change the behavior of the executable when paired with other tools such as the Scribber 2.
  • Added preference item "Post-Reset Delay" and appropriate code to allow user to specify a value other than the default 90 ms delay that occurs after reset and before host transmission. This allows the use of additional reset circuitry in applications while accommodating the increased reset signal propagation to the Propeller.
  • Moved library files to <install folder>\Library.
  • Moved library demo files to <install folder>\Library\_Demos.
  • Added menu item (Run > Parallax Serial Terminal) to launch, or show, the Parallax Serial Terminal (installed with Propeller Tool).
  • Added shortcut (F12) to launch or show Parallax Serial Terminal. This behaves like a toggle, flipping back and forth between the Propeller Tool and the Parallax Serial Terminal.
  • Added shortcut (F1) to display Propeller Help.
  • Added menu item (File > New (From Template)) to create a new edit/file with a user-settable template loaded into it. This is initially set to load the Simple_Spin_Template.spin code from the installation folder.
  • Added shortcut (Ctrl+Alt+N) to perform "New (From Template)," see above, while Ctrl+N still performs "New" item.

  • Added "New (From Template)" item to edit tab's popup menu. This behaves just like File > New (From Template).

  • Added Preference item (Preferences > Files & Folders > New File Template) to allow user to select a template for the "New (From Template)" feature.

  • Updated File > Archive feature to compile starting from the currently selected edit tab (instead of using the current project in the Object View). Also enhanced the menu item to dynamically display the target object in quotes (Archive "object_name") to clearly indicate what it will archive.

  • Added "Archive" item to edit tab's popup menu. This behaves just like File > Archive "object_name".

  • Updated "branding" in various parts of software to reflect the Parallax Semiconductor name and copyright information.
  • Added QuickStart Schematic item to Help menu.
  • Changed Help menu's "About" item to "About Propeller Tool."
  • Added "Visit Propeller Chip Discussions" item to Help menu.
  • Added "Visit Parallax Semiconductor Forum" item to Help menu.

Bug Fixes

  • Fixed bug causing exception upon application termination. Code was checking the state of certain objects that may have already been destroyed as a result of Windows messages- thus possibly reading invalid memory depending on message arrival order.
  • Fixed Help menu's Visit PE Kit Discussions link.
